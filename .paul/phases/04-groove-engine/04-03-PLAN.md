<objective>
## Goal
Each layer strip gains a template selector (5–8 named presets) and a morph knob that
continuously interpolates from neutral to the template's target Aesthetics. Morph at
100% sounds exactly as the template specifies; morph at 0% sounds unprocessed.

## Purpose
F3-AC6 (template + morph non-cosmetic) and F3-AC7 (validation recording) both require
this UI. The morph knob is LAYERZ's primary UX affordance for the "few parameters,
easily subverted" promise — one knob to control all groove intensity.

## Output
- `Source/gui/GroovePanel.h/.cpp` — template selector + morph knob for one layer
- `Source/model/Schema.h` — Template and TemplateDef types; Layer.template_name
- `Source/gui/BeatLayerStrip.h/.cpp` — GroovePanel added to each row
- `Source/gui/BassLayerStrip.h/.cpp` — GroovePanel added
</objective>

## Template naming convention (from PROJECT.md)
Each layer has 5–8 templates per genre family. Implemented as named presets in a
static table. Template application = write target Aesthetics values into
Layer.aesthetics × morph_amount. morph_amount=0 → all zeros; morph_amount=1 → full template.

    **Morph application helper** (inline in Schema.h, used by AestheticResolver):
    ```cpp
    inline Aesthetics applyMorph(const Aesthetics& neutral, const Aesthetics& target, float t) {
        // linear interpolation between neutral (t=0) and target (t=1)
        Aesthetics out;
        out.drag           = neutral.drag   + (target.drag   - neutral.drag)   * t;
        out.push           = neutral.push   + (target.push   - neutral.push)   * t;
        out.roll_mult      = (t > 0.5f) ? target.roll_mult : neutral.roll_mult;
        out.roll_vel_decay = neutral.roll_vel_decay + (target.roll_vel_decay - neutral.roll_vel_decay) * t;
        out.stutter_reps   = (t > 0.5f) ? target.stutter_reps : neutral.stutter_reps;
        out.stutter_gate   = neutral.stutter_gate + (target.stutter_gate - neutral.stutter_gate) * t;
        out.fracture_prob  = neutral.fracture_prob + (target.fracture_prob - neutral.fracture_prob) * t;
        out.fracture_seed  = target.fracture_seed;
        return out;
    }
    ```

    **BeatSequencer.process()**: before calling AestheticResolver, compute the effective
    Aesthetics using applyMorph():
    ```cpp
    // Resolve effective aesthetics from template + morph
    Aesthetics neutral{};
    const Aesthetics* target = &track.aesthetics; // default: use stored aesthetics directly
    Aesthetics effective = (layer.morph_amount > 0.0f && !layer.template_name.empty())
        ? applyMorph(neutral, findTemplate(layer.template_name, layer.type), layer.morph_amount)
        : track.aesthetics;
    // Use effective in resolver call
    ```
