  <files>Source/audio/MonoSynth.h, Source/audio/MonoSynth.cpp</files>
  <action>
    **MonoSynth.h:**
    ```cpp
    class MonoSynth {
    public:
        void prepare(const juce::dsp::ProcessSpec& spec) noexcept;

        // Audio thread: trigger note (MIDI note 0-127, velocity 0-1).
        // Mono: interrupts any currently sounding note immediately (no overlap).
        // Glide: if glide_ms > 0, pitch interpolates from previous pitch.
        void trigger(int midiNote, float velocity, const BassVoiceParams& params) noexcept;

        // Audio thread: release current note (start release phase).
        void release() noexcept;

        // Audio thread: render numSamples into buf channel 0 starting at startSample.
        // Clamps to buffer bounds. Zero allocation.
        void process(juce::AudioBuffer<float>& buf, int startSample, int numSamples) noexcept;

        bool isActive() const noexcept; // true if envelope not in idle state
    private:
        double sampleRate_ = 44100.0;
        BassVoiceParams params_;

        // Oscillator
        double phase_ = 0.0;

        // Portamento (glide)
        juce::SmoothedValue<double, juce::ValueSmoothingTypes::Linear> currentPitchHz_;

        // ADSR envelope
        juce::ADSR envelope_;
        juce::ADSR::Parameters envelopeParams_;

        // LP filter (zero-delay feedback, stable resonance)
        juce::dsp::StateVariableTPTFilter<float> filter_;
    };
    ```

    **MonoSynth.cpp:**

    `prepare()`: call currentPitchHz_.reset(sampleRate_, 0.0) to start non-smoothed;
    envelope_.setSampleRate(sampleRate_); filter_.prepare(spec); filter_.setType(SVF LP);
    filter_.setCutoffFrequency(800.0f); filter_.setResonance(0.3f).

    `trigger(midiNote, velocity, params)`:
    - Store params_
    - target_hz = 440.0 * pow(2.0, (midiNote - 69.0) / 12.0)
    - If glide_ms > 0 AND envelope was active: currentPitchHz_.setTargetValue(target_hz)
      with smoothingTime = glide_ms/1000.0 seconds (call reset with new time first)
    - Else: currentPitchHz_.setCurrentAndTargetValue(target_hz) (instant jump)
    - Update filter coefficients: filter_.setCutoffFrequency(params_.filter_cutoff),
      filter_.setResonance(params_.filter_resonance * 0.95f + 0.01f) [map 0-1 to 0.01-0.96]
    - Update envelope params: setParameters({attack, decay, sustain, release} in seconds)
    - velocity_ = velocity
    - envelope_.noteOn()

    `release()`: envelope_.noteOff()

    `process(buf, start, numSamples)`:
    - If not isActive() return
    - For each sample:
      - Advance glide: double hz = currentPitchHz_.getNextValue()
      - Advance oscillator: phase_ += hz / sampleRate_; if >= 1.0 phase_ -= 1.0
      - Generate waveform:
        - SAW: sample = static_cast<float>(phase_ * 2.0 - 1.0)  (bipolar -1..1)
        - SQUARE: sample = phase_ < 0.5f ? 1.0f : -1.0f
      - Apply envelope: float env = envelope_.getNextSample()
      - Apply filter: float filtered = filter_.processSample(0, sample * env)
        (channel index 0 for mono filter)
      - Write: buf[0][start+i] += filtered * velocity_ * params_.volume * 0.6f
        (0.6 scale matches drum voice levels to avoid clip)
    - Clamp write count to juce::jmin(numSamples, buf.getNumSamples() - start)

    Avoid: calling envelope_.setSampleRate() inside process (sets up allocations).
    Avoid: calling filter_.prepare() inside process.
  </action>

    // Find BASS layer (if any) in pattern.layers
    for (const auto& layer : pattern.layers) {
        if (layer.type != LayerType::BASS) continue;
        // For each beat event: find matching step in Layer.events[]
        for (int bi = 0; bi < beatEvents.count; ++bi) {
            const auto& beatEv = beatEvents.data[bi];
            int beat_step = beatEv.beat_index % pattern.length_steps;
            for (const auto& event : layer.events) {
                if (event.step == beat_step && event.midi_note >= 0) {
                    // Auto gate: release previous note, trigger new one
                    bank.releaseBass();

        }
