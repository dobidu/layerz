    **PitchedStepRow.h/.cpp:**
    Extends the StepRow concept for pitched BASS events.

    ```cpp
    class PitchedStepRow : public juce::Component {
    public:
        PitchedStepRow(ProjectStore& store, int layerIndex = 0);
        void resized() override;
        void updateFromSnapshot(const Project& snap);
        void setCurrentStep(int step) noexcept;
    private:
        ProjectStore& store_;
        int layerIndex_ = 0;
        int currentStep_ = -1;
        juce::TextButton stepButtons_[16];
    };
    ```

    **onClick per button i:**
    1. Read snapshot: find the BASS layer in patterns[active_pattern_index].layers
    2. If event exists at step i: REMOVE it (erase with remove_if like StepRow)
    3. If not exists: show a `juce::PopupMenu` with note options:
       - C1(24), D1(26), E1(28), F1(29), G1(31), A1(33), B1(35),
         C2(36), D2(38), E2(40), F2(41), G2(43), A2(45), Bb2(46), B2(47),
         C3(48) (16 common bass notes)
       - Menu item text: "C2 (36)", "E2 (40)" etc.
       - On selection: postMutation to add Event with step=i, midi_note=selected, velocity=0.8
    4. Optimistic update: if adding, immediately set button text to note name

    **Button visual:**
    - Inactive (no event): dark grey, no label
    - Active (event exists): amber fill (0xFFE67E22), label = note name (e.g. "G2")
      Note name from midi_note: `noteNames[midi_note % 12] + juce::String(midi_note/12 - 1)`
    - Current step indicator: 3px amber bar at bottom (leave 4px gap in resized())

    **updateFromSnapshot():** read BASS layer events, update each button state.
    Use juce::dontSendNotification on all button state changes.

    **BassVoiceParamPanel.h/.cpp:**
    Controls for BassVoiceParams. Similar structure to VoiceParamPanel.
    Members:
    - `juce::Slider filterCutoff_`   — range 80–8000Hz, exponential, log scale
    - `juce::Slider filterRes_`      — range 0–1
    - `juce::Slider attackSlider_`   — range 1–500ms
    - `juce::Slider decaySlider_`    — range 1–500ms
    - `juce::Slider sustainSlider_`  — range 0–1
    - `juce::Slider releaseSlider_`  — range 1–500ms
    - `juce::Slider glideSlider_`    — range 0–500ms
    - `juce::TextButton waveformBtn_` — toggle SAW/SQR

    All onChange: postMutation to update BassVoiceParams in BASS layer.
    All updateFromSnapshot(): setValue/setToggleState with juce::dontSendNotification.

    Layout: 2-column grid. Labels above each slider. Waveform toggle spans full width.
    Filter cutoff use logarithmic slider: `filterCutoff_.setSkewFactorFromMidPoint(800.0)`.
